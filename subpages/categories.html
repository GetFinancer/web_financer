KategorienCategories

Kategorien helfen dir, deine Transaktionen zu ordnen. Du kannst eigene Kategorien für Einnahmen und Ausgaben erstellen und sie hierarchisch strukturieren.

Categories help you organize your transactions. You can create custom categories for income and expenses and structure them hierarchically.

KategorietypenCategory Types

Es gibt zwei Typen von Kategorien:

There are two types of categories:

  • Ausgabe-Kategorien – z. B. Miete, Lebensmittel, Transport, Freizeit
  • Einnahme-Kategorien – z. B. Gehalt, Freelancing, Erstattungen
  • Expense categories – e.g. rent, groceries, transport, leisure
  • Income categories – e.g. salary, freelancing, refunds

Beim Erstellen einer Transaktion werden dir nur die passenden Kategorien angezeigt (Einnahme-Kategorien bei Einnahmen, Ausgabe-Kategorien bei Ausgaben).

When creating a transaction, only the matching categories are shown (income categories for income, expense categories for expenses).

Neue Kategorie erstellenCreate New Category

  1. Navigiere zur Seite „Kategorien“.
  2. Klicke auf „Neue Kategorie“.
  3. Gib einen Namen ein.
  4. Wähle den Typ (Einnahme oder Ausgabe).
  5. Wähle eine Farbe – diese wird in den Auswertungen und Diagrammen verwendet.
  6. Optional: Wähle eine Oberkategorie für hierarchische Strukturierung.
  7. Klicke auf „Speichern“.
  1. Navigate to the "Categories" page.
  2. Click "New category".
  3. Enter a name.
  4. Select the type (income or expense).
  5. Select a color – this is used in analytics and charts.
  6. Optional: Select a parent category for hierarchical structure.
  7. Click "Save".

Hierarchische KategorienHierarchical Categories

Du kannst Unterkategorien erstellen, indem du einer Kategorie eine Oberkategorie zuweist. Beispiel:

You can create subcategories by assigning a parent category. Example:

  • Wohnen (Oberkategorie)
    • Miete
    • Strom
    • Internet
  • Transport (Oberkategorie)
    • ÖPNV
    • Tanken
    • KFZ-Versicherung
  • Housing (parent category)
    • Rent
    • Electricity
    • Internet
  • Transport (parent category)
    • Public transit
    • Fuel
    • Car insurance

In den Auswertungen kannst du nach Oberkategorien filtern, um eine zusammengefasste Ansicht zu erhalten.

In the Analytics, you can filter by parent categories to get a summarized view.

FarbenColors

Jede Kategorie hat eine eigene Farbe. Diese wird in Diagrammen und Übersichten verwendet, um Kategorien visuell voneinander zu unterscheiden. Wähle möglichst unterschiedliche Farben für bessere Lesbarkeit.

Each category has its own color. This is used in charts and overviews to visually distinguish categories from each other. Choose different colors for better readability.

Kategorie bearbeiten oder löschenEdit or Delete Category

Klicke auf eine bestehende Kategorie, um sie zu bearbeiten. Du kannst Name, Farbe und Oberkategorie ändern.

Click on an existing category to edit it. You can change the name, color, and parent category.

Hinweis

Kategorien, die bereits von Transaktionen oder Daueraufträgen verwendet werden, sollten nur mit Bedacht gelöscht werden. Die zugehörigen Buchungen verlieren ihre Kategoriezuordnung.

Note

Categories that are already used by transactions or recurring entries should only be deleted with care. The associated entries will lose their category assignment.
