DaueraufträgeRecurring Transactions

Daueraufträge sind wiederkehrende Buchungen, die automatisch als geplante Transaktionen im Dashboard erscheinen. Ideal für Miete, Gehalt, Abos und andere regelmäßige Zahlungen.

Recurring transactions are repeating entries that automatically appear as planned transactions on the dashboard. Ideal for rent, salary, subscriptions, and other regular payments.

Neuen Dauerauftrag erstellenCreate New Recurring Transaction

  1. Navigiere zur Seite „Daueraufträge“ über das Menü.
  2. Klicke auf „Neuer Dauerauftrag“.
  3. Gib einen Namen ein (z. B. „Miete“, „Netflix“).
  4. Wähle den Typ (Einnahme oder Ausgabe).
  5. Gib den Betrag ein.
  6. Wähle die Frequenz und das Startdatum.
  7. Wähle Konto und Kategorie.
  8. Optional: Setze ein Enddatum, falls der Dauerauftrag zeitlich begrenzt ist.
  9. Klicke auf „Speichern“.
  1. Navigate to the "Recurring" page via the menu.
  2. Click "New recurring transaction".
  3. Enter a name (e.g. "Rent", "Netflix").
  4. Select the type (income or expense).
  5. Enter the amount.
  6. Select the frequency and start date.
  7. Select account and category.
  8. Optional: Set an end date if the recurring transaction is time-limited.
  9. Click "Save".

AusnahmenExceptions

Manchmal weicht ein einzelner Monat vom Standard ab – z. B. eine höhere Stromrechnung oder ein ausgesetzter Beitrag. Dafür gibt es Ausnahmen.

Sometimes a single month deviates from the standard – e.g. a higher electricity bill or a skipped payment. Exceptions handle this.

Betrag für eine Einzelbuchung ändernChange Amount for a Single Entry

  1. Gehe zum Dashboard und finde die geplante Transaktion.
  2. Klicke darauf und wähle „Betrag ändern“.
  3. Gib den abweichenden Betrag ein.
  4. Wähle, ob die Änderung nur für diesen Monat oder für alle zukünftigen gelten soll.
  1. Go to the Dashboard and find the planned transaction.
  2. Click on it and select "Change amount".
  3. Enter the different amount.
  4. Choose whether the change applies only to this month or to all future entries.

Einzelbuchung überspringenSkip a Single Entry

Du kannst eine einzelne Buchung eines Dauerauftrags überspringen, ohne den gesamten Dauerauftrag zu deaktivieren.

You can skip a single entry of a recurring transaction without deactivating the entire recurring transaction.

Ausnahme zurücksetzenReset Exception

Wenn du eine Ausnahme rückgängig machen willst, klicke erneut auf die geplante Transaktion und wähle „Zurücksetzen“. Der Originalbetrag wird wiederhergestellt.

If you want to undo an exception, click on the planned transaction again and select "Reset". The original amount will be restored.

Dauerauftrag bearbeiten oder löschenEdit or Delete Recurring Transaction

Auf der Daueraufträge-Seite kannst du bestehende Einträge bearbeiten (Name, Betrag, Frequenz etc.) oder vollständig löschen. Beim Löschen werden auch alle zugehörigen geplanten Transaktionen entfernt.

On the recurring page, you can edit existing entries (name, amount, frequency, etc.) or delete them entirely. Deleting also removes all associated planned transactions.

Tipp

Nutze Daueraufträge auch für geplante Einnahmen wie dein Gehalt. So siehst du auf dem Dashboard, wie viel Budget du im laufenden Monat noch zur Verfügung hast.

Tip

Use recurring transactions also for planned income like your salary. This way you can see on the dashboard how much budget you have left for the current month.
