TransaktionenTransactions

Auf der Transaktionsseite siehst du alle Buchungen und kannst neue erstellen, bestehende bearbeiten oder löschen.

On the transactions page you can see all entries and create new ones, edit existing ones, or delete them.

TransaktionstypenTransaction Types

Jede Transaktion hat einen von drei Typen:

Each transaction has one of three types:

TypBeschreibung
EinnahmeGeld kommt rein (z. B. Gehalt, Erstattungen).
AusgabeGeld geht raus (z. B. Einkauf, Miete).
UmbuchungGeld wird zwischen zwei eigenen Konten verschoben.
TypeDescription
IncomeMoney coming in (e.g. salary, refunds).
ExpenseMoney going out (e.g. shopping, rent).
TransferMoney moved between your own accounts.

Neue Transaktion erstellenCreate New Transaction

  1. Klicke auf den + Button oben rechts oder nutze den schnellen Hinzufügen-Button auf dem Dashboard.
  2. Wähle den Typ (Einnahme, Ausgabe oder Umbuchung).
  3. Gib den Betrag ein.
  4. Wähle das Konto, auf das die Buchung geht.
  5. Wähle eine Kategorie für die Zuordnung.
  6. Optional: Gib eine Beschreibung ein und wähle ein Datum.
  7. Klicke auf „Speichern“.
  1. Click the + button in the top right or use the quick add button on the dashboard.
  2. Select the type (income, expense, or transfer).
  3. Enter the amount.
  4. Select the account for the transaction.
  5. Select a category for classification.
  6. Optional: Enter a description and select a date.
  7. Click "Save".
Tipp

Bei einer Umbuchung wählst du ein Quell- und ein Zielkonto. Der Betrag wird vom Quellkonto abgezogen und dem Zielkonto gutgeschrieben.

Tip

For a transfer, you select a source and a destination account. The amount is deducted from the source account and credited to the destination account.

Transaktionen filternFilter Transactions

Über die Filter am oberen Rand der Liste kannst du die angezeigten Transaktionen einschränken:

Using the filters at the top of the list, you can narrow down the displayed transactions:

  • Zeitraum – Wähle einen Datumsbereich.
  • Konto – Zeige nur Buchungen eines bestimmten Kontos.
  • Kategorie – Filtere nach einer Kategorie.
  • Typ – Nur Einnahmen, nur Ausgaben oder nur Umbuchungen.
  • Period – Select a date range.
  • Account – Show only entries for a specific account.
  • Category – Filter by category.
  • Type – Only income, only expenses, or only transfers.

Transaktion bearbeitenEdit Transaction

Klicke auf eine Transaktion in der Liste, um sie zu bearbeiten. Ändere die gewünschten Felder und klicke auf „Speichern“.

Click on a transaction in the list to edit it. Change the desired fields and click "Save".

Transaktion löschenDelete Transaction

Beim Bearbeiten einer Transaktion findest du unten die Option zum Löschen. Gelöschte Transaktionen können nicht wiederhergestellt werden.

When editing a transaction, you will find the delete option at the bottom. Deleted transactions cannot be restored.

Hinweis

Das Löschen einer Transaktion beeinflusst sofort den Kontostand des zugehörigen Kontos.

Note

Deleting a transaction immediately affects the balance of the associated account.
