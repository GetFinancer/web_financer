Was ist Financer?What is Financer?

Financer ist eine leichtgewichtige, selbstgehostete Web-App zur Verwaltung deiner persönlichen Finanzen. Sie läuft als Progressive Web App (PWA) und kann wie eine native App auf deinem Smartphone installiert werden.

Financer is a lightweight, self-hosted web app for managing your personal finances. It runs as a Progressive Web App (PWA) and can be installed on your smartphone just like a native app.
